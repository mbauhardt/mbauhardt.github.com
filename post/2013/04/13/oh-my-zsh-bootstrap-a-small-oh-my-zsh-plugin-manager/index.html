Oh My Zsh Bootstrap, A Small Oh My Zsh Plugin Manager

April 13, 2013

I started a small but helpful project oh-my-zsh-bootstrap to maintain my oh-my-zsh. Oh My Zsh Bootstrap is plugin for Oh My Zsh which provides commands to maintain your plugins and themes

  • enable_plugin
  • disable_plugin
  • download_plugin

  • update_plugin

  • enable_theme

  • download_theme

  • update_theme

You can install Oh My Zsh Bootstrap with a onliner

curl -L https://github.com/mbauhardt/oh-my-zsh-bootstrap/raw/master/tools/install.zsh | zsh
